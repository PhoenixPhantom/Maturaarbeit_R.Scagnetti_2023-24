# Folders


<pre>
├───.idea
│ 		Editor generated files;
├───Config
│ 		In-Engine Settings; Can be modified through toggles and dropdowns in-engine.
├───Content
│ 		All assets used in the Engine; This is also the "Content" folder in the engine filebrowser. All files not listed [here](README.md#UsedAssets) contain custom logic and configurations.
├───Platforms
│ 		Auto generated engine content;

└───Source
	All files concerning user added functionality using C++; This is also the "C++" folder in the engine filebrowser. The .cs files are automatically generated by UE5 as a list of the modules used in compilation. While as all .h and .cpp files contain custom logic and setup. 
</pre>
